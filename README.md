# 日野町 オープンデータ・ダッシュボード (プロトタイプ)

滋賀県日野町の統計資料「統計でわかる！日野のすがた」を、視覚的にわかりやすく閲覧・2次利用できるようにしたシビックテックのプロトタイプダッシュボードです。

ブラウザ上で動作するSPA（シングルページアプリケーション）として構築されており、デジタル庁のダッシュボードデザインガイドラインを参考にした、認知負荷の低いUI/UXを目指しています。

## 📊 現状格納しているデータの種類

現在、以下の6つの統計データをCSV化し、ダッシュボードに組み込んでいます。
各データは `data/` ディレクトリ内にCSV形式で格納されており、自由にダウンロードして2次利用が可能です。

1. **人口・世帯数** (`hino_population.csv`) - 総人口（男女別）および世帯数の推移
2. **年齢3区分別人口** (`hino_age.csv`) - 年少・生産年齢・老年人口の割合変化
3. **産業区分別就業者数** (`hino_industry.csv`) - 第一次〜第三次産業における就業者数の推移
4. **地目別面積** (`hino_land_area.csv`) - 田、畑、宅地、林山などの面積推移
5. **観光客入込数** (`hino_tourism.csv`) - 日帰り・宿泊別の観光客数の動向
6. **ごみ投入量** (`hino_garbage.csv`) - 可燃物および不燃物の排出量トレンド

## 💡 iframeでのサイト埋め込みについて

本ダッシュボードは、他のウェブサイトへの埋め込み表示に対応しています。
URLの末尾に `?embed=true` を付与することで、ヘッダーや余白が非表示になり、iframe内に最適化された状態で表示されます。

```html
<iframe 
    src="https://kken78.github.io/hino-dashboard/?embed=true#population" 
    width="100%" 
    height="500" 
    style="border:none; background:transparent;" 
    scrolling="no">
</iframe>
