# 日野のすがた｜ダッシュボード（プロトタイプ）

滋賀県日野町の公式統計資料「統計でわかる！日野のすがた」のデータを抽出し、視覚的にわかりやすく閲覧できるように構築したオープンデータ利活用のための実証実験プロジェクトです。

ブラウザ上で軽快に動作するSPA（シングルページアプリケーション）を採用し、デジタル庁の「ダッシュボードデザインガイドブック」に準拠した認知負荷の低いUI/UXを実装しています。

## 🚀 本プロトタイプの主な特徴

1. **人口を軸にした9つのハイライト指標**
   * トップページは「まちの今を読み解く」観点で、**出生数・高齢化率・社会増減**を先頭に、世帯人員・外国籍住民・観光・ごみ・鉄道・財政を続ける構成にしています。数値はCSVから自動算出し、各カードに直近の推移を示すスパークライン（折れ線）を併記します。
2. **指標ごとの「良し悪しの向き」を明示**
   * 指標によって「増えると良い」「減ると良い」「方向では判断しない（中立）」が異なります。各カードに向きを文章で明記し、改善方向は緑、注意方向は赤、中立は灰で表示。色覚特性に配慮し、色だけでなく▲▼記号と説明文でも伝わるようにしています。
3. **公式統計の値を尊重した算出**
   * 1人あたりごみ排出量は独自計算ではなく、公式資料が公表する「1人当り（kg）」の確定値をそのまま採用します。1日平均（観光・鉄道）は、対象年次が平年（365日）か閏年（366日）か、暦年か会計年度（2月の閏日を含むか）かをプログラムが自動判定して厳密に割り算します。
4. **データの出所を可視化**
   * 各ハイライトに「国勢」「業務」のタグを付け、5年周期の国勢調査（最新確定値＝令和2年）と毎年更新の町業務統計（令和6年）の違いがひと目で分かるようにしています。
5. **自動巡回・連続移動・埋め込み対応**
   * 全グラフを5秒間隔で切り替えるスライドショー（庁舎サイネージや会議向け）、「前／次の統計」ボタン、`?embed=true` による外部サイト埋め込みに対応。閲覧中はグラフをクリックすると自動巡回が止まります。
6. **モバイル・アクセシビリティ対応**
   * スマートフォンでは1列レイアウトで大きく表示。キーボード操作時のフォーカス表示、`prefers-reduced-motion`（視差効果を減らす設定）にも対応しています。

## 📊 収録データ一覧

現在、公式資料の目次構成に沿って、以下の28種のデータをCSV形式で `data/` フォルダ内に公開しています。

### 1. 位置・地勢・気象
* **地目別面積** (`hino_land_area.csv`) - 田、畑、宅地、林山などの面積推移
* **気象概況** (`hino_weather.csv`) - 月別平均気温・降水量

### 2. 人口
* **人口・世帯数** (`hino_population.csv`) - 総人口（男女別）および世帯数の推移
* **人口動態** (`hino_vital_stats.csv`) - 出生・死亡・自然増加、転入・転出・社会増加の推移
* **外国人登録者数** (`hino_foreigners.csv`) - 国籍別（ブラジル、フィリピン、中国、韓国・朝鮮、ペルー、その他）の登録者数推移
* **産業区分別就業者数** (`hino_industry.csv`) - 第一次〜第三次産業における就業者数の推移
* **年齢3区分別人口** (`hino_age.csv`) - 年少・生産年齢・老年人口の構成と高齢化率（％）の推移

### 3. 産業
* **観光客入込数** (`hino_tourism.csv`) - 日帰り・宿泊別の観光客数の動向
* **工業の推移** (`hino_industry_trend.csv`) - 工業事業所数および従業者数の推移

### 4. 農林業
* **経営耕地面積** (`hino_farming_area.csv`) - 田、畑、樹園地の面積推移
* **農地転用状況** (`hino_farmland_conversion.csv`) - 住宅用地や駐車場などへの転用面積（㎡）推移

### 5. 運輸・交通
* **近江鉄道日野駅乗降客数** (`hino_railway.csv`) - 日野駅の年間乗降客数の推移
* **町営バス利用状況** (`hino_bus.csv`) - 定期・定期外利用者数の推移
* **交通事故発生状況** (`hino_traffic_accident.csv`) - 町内における交通事故件数の推移

### 6. 水道・通信・消費
* **上水道給水状況** (`hino_water_supply.csv`) - 給水人口と年間総配水量の推移
* **公共下水道整備状況** (`hino_sewerage.csv`) - 下水道の普及率と水洗化率の推移
* **ごみ投入量** (`hino_garbage.csv`) - 可燃物・不燃物の投入量、および公式統計に基づく1人あたり排出量（kg）
* **たばこ消費量** (`hino_tobacco.csv`) - 町内のたばこ年間消費量（千本）の推移 ※新規

### 7. 社会保障
* **国民健康保険** (`hino_nhis.csv`) - 国保被保険者数と加入率の推移
* **後期高齢者医療** (`hino_latter_elderly.csv`) - 受給者数・給付件数・1人あたり費用額の推移 ※新規
* **介護保険（被保険者）** (`hino_nursing_care.csv`) - 第1号被保険者数（前期・後期別）の推移
* **要介護認定者数** (`hino_care_certification.csv`) - 要支援1〜要介護5の段階別認定者数の推移 ※新規
* **保育所・認定こども園** (`hino_childcare.csv`) - 定員および在籍人員の推移

### 8. 保健衛生
* **三大疾病別死亡数** (`hino_diseases.csv`) - がん・心疾患・脳血管疾患による死亡数推移

### 9. 教育・文化
* **図書館の利用状況** (`hino_library.csv`) - 図書館の来館者数と利用冊数の推移
* **文化施設入館者数** (`hino_cultural_facilities.csv`) - 商人館・ふるさと館・感応館の入館者数推移

### 10. 財政
* **決算分析指数** (`hino_finance_index.csv`) - 経常収支比率などの健全化指標

### 11. 治安・消防
* **火災・救急出動** (`hino_fire_ambulance.csv`) - 火災発生件数と救急車出動件数

* **出典:** [日野町公式ウェブサイト「統計でわかる！日野のすがた」](https://www.town.shiga-hino.lg.jp/0000007518.html)

## 💡 外部サイトへの埋め込み

本ダッシュボードは、他のウェブサイトやブログへの埋め込み（iframe）に対応しています。URLの末尾に `?embed=true` を付与すると、ヘッダーや余白が非表示になり、最適化された状態で表示されます。

    <iframe 
        src="https://kken78.github.io/hino-dashboard/?embed=true#population" 
        width="100%" 
        height="500" 
        style="border:none; background:transparent;" 
        scrolling="no">
    </iframe>

※ `#population` の部分を `#tourism` などに変更すると、最初から指定したグラフを開いた状態で表示できます。

## 🔄 データの更新方針

* **令和7年（2025年）国勢調査について:** 人口・世帯・年齢・産業就業者・外国人就業などの指標は、5年に1度の国勢調査の確定値に依拠しています。本ダッシュボードの該当指標は現時点で**令和2年（2020年）**が最新確定値です。令和7年調査の結果が公表され次第、`hino_population.csv` / `hino_age.csv` / `hino_industry.csv` などを差し替える予定です。トップの注記と各カードの「国勢」タグで、いつ時点のデータかが分かるようにしています。
* **業務・決算統計:** 出生・社会増減・ごみ・鉄道・財政などは毎年更新されるため、原則として直近年（令和6年・令和6年度）の確定値を採用しています。

## 🛠️ データ修正・拡充の記録（プロトタイプ第2版）

公式資料（「統計でわかる！日野のすがた2025」）の各統計表と全CSVを突き合わせ、以下を実施しました。

* **ごみ投入量の構造を修正:** 旧版は1人あたり排出量をアプリ側で概算していましたが、公式資料が「1人当り（kg）」の確定値を掲載しているため、その値を直接採用する形に変更しました（可燃物・不燃物の投入量、し尿量の列も追加）。
* **人口動態に社会増減を追加:** 自然増加・社会増加（転入−転出）の列を追加し、ハイライト指標として活用できるようにしました。
* **年齢区分に高齢化率を追加:** 公式の構成比（％）を高齢化率として列追加しました。
* **新規データ3種を追加:** 後期高齢者医療、要介護認定者数、たばこ消費量。
* **その他:** 表記の誤りを修正し、グラフ・指標の見せ方を改善しました。
